Phase 31G — Stove Control Polish

What changed:
- Shift + right-click with an empty hand now toggles the stove burner on/off when cookware is mounted.
- Shift + right-click while holding the same cookware item will remove the pan/pot from an empty, cooled stove.
- Gas flame animation was sped up and given stronger flicker so it feels closer to a live burner instead of a slow bob.

Notes:
- Normal right-click still supports placing cookware, loading ingredients, and pulling output.
- This phase is intentionally narrow so it can be backported cleanly onto the current working base.
