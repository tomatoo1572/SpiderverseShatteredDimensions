- 8 Cobblestone + Coal center -> Stove

## Notes
- This is the **foundation pass**, not the full final cooking expansion.
- Crops are items for now; crop planting / growth was not added yet.
- Pot and Pan are inventory items for now and can be wired into later station passes.
- The existing furnace UI is reused as the processing UI for the new cooking stations.
