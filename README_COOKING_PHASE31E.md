Phase 31E - Appliance visual polish

What changed:
- Stove now has a visible on/off indicator built into the block-space visual.
- Burner state is shown with:
  - a front indicator lamp
  - a rotated burner knob
  - a glowing burner ring when the stove is on
- Pan and pot visuals were rebuilt to feel deeper and less flat.
- Pan now has:
  - thicker base
  - raised walls
  - visible inner cavity
  - rim lip
  - better handle/grip shape
- Pot now has:
  - taller outer body
  - visible inner cavity depth
  - top rim lip
  - side handles

Current limitations:
- These are still procedural in-world visuals, not imported custom models yet.
- Oven is still more UI-driven than the stove.
- Food on cookware still uses icon billboards for now.
