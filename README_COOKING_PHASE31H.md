# SSD Phase 31H - Crops and Ingredient Production

Added in this pass:
- Farmland block
- Plantable wheat, carrot, potato, rice, tomato, and cucumber crops
- Plantable strawberry, blueberry, and blackberry bushes
- Mango planting now creates a mango sapling that grows into a small mango tree
- Random crop growth tick in loaded chunks
- Right-click harvest with empty hand on mature crops and bushes
- Wheat ingredient item added to the Food tab
- Prep Table recipe: Wheat -> Flour

Planting rules:
- Wheat, carrot, potato, rice, tomato, cucumber:
  - right-click the ground with the ingredient in hand
  - grass/dirt auto-tills into farmland
- Strawberry, blueberry, blackberry:
  - right-click grass/dirt/farmland to place bush
- Mango:
  - right-click grass/dirt/farmland with a mango to plant a sapling

Harvesting:
- Empty hand + right-click mature crops/bushes to harvest
- Tomato/cucumber/berry bushes regrow after harvest
- Wheat/carrot/potato/rice harvest clears the crop and leaves farmland
