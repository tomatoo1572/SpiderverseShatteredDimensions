# Spiderverse Shattered Dimensions - Phase 3 Player Core

This build includes:
- smoother block break/place feel with instant click and held-repeat interaction
- Minecraft-style 9-slot hotbar foundation
- middle-mouse pick block into the hotbar
- health + stamina component foundation
- stamina drain / regen tied to sprinting
- simple status bars HUD
- void rescue if the player falls far below the world

## Controls
- WASD: move
- Space: jump
- Shift: sprint
- F: toggle fly
- Esc: release/capture mouse
- R: regenerate world and re-prime spawn
- Left Mouse: break block
- Right Mouse: place selected block
- Middle Mouse: pick targeted block into hotbar
- 1-9: hotbar select
- Mouse Wheel: cycle hotbar
