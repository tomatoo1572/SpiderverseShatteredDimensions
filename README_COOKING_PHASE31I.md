Phase 31I - Multi-Ingredient Pot Cooking + Crop Growth Command

Added:
- Stove now supports a second ingredient for pot recipes.
- Pot recipes added:
  - Broth + Noodles -> Chicken Noodle Soup
  - Noodles + Shredded Cheese -> Mac and Cheese
- Existing one-ingredient pan/pot recipes remain intact.
- Crouch + empty-hand on stove now pulls the second ingredient out first, then the primary ingredient.
- Added crop growth speed command:
  - /cropgrow <multiplier>
  - /cropgrow <multiplier> <bonus_ticks>
Examples:
- /cropgrow 4
- /cropgrow 8 20

Notes:
- multiplier speeds regular loaded-chunk crop growth ticks
- bonus_ticks forces extra immediate growth attempts across loaded chunks
